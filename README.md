# Topwrap Utils

Copyright (c) 2026 [Antmicro](https://www.antmicro.com)

This repository is a collection of utilities (tools or plugins) for [topwrap](https://github.com/antmicro/topwrap).
Each utility is distributed as a python package.

## Installation 

Installation instructions are provided in the documentation of each utility.
In general, to install a python-package from this repository, you would use the following command

```bash
pip3 install "git+https://github.com/antmicro/topwrap-utils.git#subdirectory=UTILITY"
```

where `UTILITY` should be replaced with the name of the subdirectory containing the tool/plugin
